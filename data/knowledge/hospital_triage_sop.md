# Hospital Triage SOP (Simplified)

1. Collect chief complaint, age, sex, and full basic vital signs.
2. If red-flag pattern is present, trigger emergency workflow immediately.
3. For pediatric abdominal pain, default to pediatrics unless emergency override.
4. All triage decisions must produce a structured handover sheet.
5. Keep audit logs: input snapshot, rule matches, evidence sources, final output.
